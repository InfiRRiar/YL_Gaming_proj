Проект

Цель проекта - создание мини-сайта, который теоретически может быть компании, занимащейся разработкой игр. Надо запустить 2 файла - run.py и VKbot.py.

О сайте

Некоторые функции доступны только при наличии аккаунта. У аккаунта 2 уровня - разработчик и обычный пользователь. Уровень определяется в БД булевским значением is_developer. Эта переменная по умолчанию False, ее можно сменить только в самой БД.

У обычного пользователя в меню есть 4 основные вкладки, не считаю этой:

  • Новости - список опубликованных компанией новостей. Чтобы открыть новость полностью, надо войти в профиль
  • Проекты - список игр комапании. Если пользователь зашел в аккаунт, то он может добавить игру в свою библиотеку. Добавленные игры можно увидеть в профиле. Из своей библиотеки пользователь может скачать игру, но так как это не настоящие игры, то там просто ссылки на информацию о них)
  • Поддержка - в разделе размещена форма, которую пользователь может использовать, если нашел ошибку в работе сайта
  • Профиль - статистика пользователя: его id, информация о привязанном профиле ВК, уровень его аккаунта. Через эту вкладку пользователь также может сменить пароль и привязанный ВК. В профиле также находятся игры, которые были добавлены в библиотеку

У администратора вместо вкладки "Поддержка" есть "Сообщения об ошибках", где он может увидеть все сообщение об ошибках, которые отправляли пользователи. Также у него есть вкладка "Добавить", где администратор может добавить новую новость или проект.

Бот

Бот написан в файле VKBot.py, он нужен для привязки аккаунта ВКонтакте к профилю, а также для восстановления пароля. После регестрации появляется инструкция по привязке аккаунта. Если пользователь забыл свой пароль, то он может написать боту сообщение "сброс пароля", и если аккаунт, с которого писали это сообщение , привязан к какому-то профилю, то пароль у этого у этого профиля сменится на случайно сгенерированный, и бот напишет этот пароль, который можно использовать для входа в аккаунт и смены пароля на свой.
